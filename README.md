# Experiments with https://github.com/PolymerLabs/lit-html

## Prepare

```
  npm i
```

## Develop

```
  npm run start:dev
```

## Build
```
  npm run build
```

## Deploy

```
  npm run deploy
```

Open link: https://baden.github.io/lit-html-testing/
